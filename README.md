# Perfect Visuals
A client-side mod for Minecraft 1.21 designed to enhance PvP experience and rendering capabilities.

## Features
- **Combat Assistance:** Automated tools for tracking targets.
- **Visual Enhancements:** Customizable HUD, particles, and shaders.
- **Performance:** Optimized rendering.
